To run Customer Code:
  Download the entire Customer Folder
  Run page1.html file to view what the customers see when they come to the restuarant to reserve a table for their group.
  Run home.html file to view what the customers see when they sit as their table to order.
  Currently the website does not support a database therefore some of the functionalities are currently not working.
  The SOS option, Adding items and placing  the order are currently in the works after the database is complete.
  Be sure to enable flash for the games at the games tab to run on your browser.
